# UrbanMart Sales Dashboard

## Project Overview
A Python + Streamlit mini-project to analyze UrbanMart’s retail sales data.

## Deliverables
- `app.py` → Streamlit dashboard
- `urbanmart_sales.csv` → dataset
- `requirements.txt` → dependencies
- Reflection note

## Reflection Answers
1. **Highest revenue store location:** Downtown (based on sample data).
2. **Channel comparison:** In-store generates more revenue in this sample.
3. **Top 3 categories:** Beverages, Snacks, Personal Care.
4. **Suggested feature:** Add filter by customer segment to analyze loyalty impact.
